# Reports
This repository contains a full Open Source Intelligence (OSINT) investigation conducted on tesla.com as part of a cybersecurity learning project.

The investigation includes:

DNS Enumeration

Shodan Analysis

SecurityTrails Subdomain Review

WHOIS Data

TXT / SPF / SaaS verification review

Social Media & Public Exposure OSINT

Risk summary and recommendations

The final OSINT report is provided as a PDF file in this repository.
